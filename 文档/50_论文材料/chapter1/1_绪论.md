# 第一章 绪论

## 1.1 研究背景与意义

物联网设备规模的持续扩大催生了大量计算密集且时延敏感的应用，如目标识别、环境三维建模和高清视频分析。终端设备受限于体积与电池容量，难以独立承载此类任务。移动边缘计算（Mobile Edge Computing, MEC）将服务器下沉至网络边缘，使终端可就近获取算力支持，但地面固定基站在偏远地区、灾害现场和临时通信热点中部署成本高、灵活性差，覆盖盲区难以避免。旋翼无人机（Unmanned Aerial Vehicle, UAV）部署快、机动性强、对地视距链路质量稳定。利用多架 UAV 搭载计算载荷在固定高度飞行，为地面终端提供辅助边缘计算服务，已被广泛认为是应对动态场景算力瓶颈的有效方案。

UAV 辅助边缘计算系统的任务处理伴随双向数据流：终端先将任务数据上行传输至 UAV，UAV 完成计算后再将结果下行回传。上行数据量通常远大于下行数据量，形成显著的上下行不对称性。与此同时，UAV 飞行轨迹直接决定空地信道质量，CPU 频率分配影响计算速率，而离散的卸载决策又约束了前两者的有效范围。三类变量——二元卸载决策 $\mathbf{x}$、CPU 频率分配 $\mathbf{f}$ 和 UAV 轨迹 $\mathbf{q}$——在时空维度上相互耦合。以最小化归一化时延与能耗的加权系统成本为目标联合优化这三类变量，构成混合整数非线性规划（MINLP）问题。

传统方法处理此类问题时面临两方面瓶颈。一方面，基于数学规划的交替优化方法需要将离散变量松弛为连续值，对凸性假设要求严格，求解质量受限。另一方面，遗传算法或和声搜索等元启发式方法虽适用范围广，但搜索方向依赖人工设定的适应度函数，面对双向数据流带来的非线性时空耦合时，固定规则的搜索效率迅速下降。近年来大语言模型（Large Language Model, LLM）在零样本推理与代码生成上取得突破，NeurIPS 2025 的一项工作[^1]证明 LLM 可作为元启发式算法中的高级变异算子，自动演化出更优的目标函数。然而，现有 LLM 引导优化研究集中在网约车调度等纯离散场景，尚未涉及含非凸物理公式的通信网络。本文将该范式引入 UAV-MEC 系统，探索 LLM 的代码表征能力在含非凸物理约束的通信优化问题中的适用性。

## 1.2 国内外研究现状概述

UAV 辅助边缘计算的联合优化研究沿两条路线展开。数学规划方面，块坐标下降（BCD）、连续凸近似（SCA）和拉格朗日对偶等方法被用于将原问题分解为子问题交替求解，对连续变量具有确定的收敛轨迹，但在处理包含二元卸载决策的高维 MINLP 时，变量松弛会损害解的质量。启发式搜索方面，群智能算法虽能避免凸性假设，但其目标函数与变异算子均需专家手工设定。面对旋翼推进能耗与信道衰落等底层物理机制，固定的人工规则在双向数据流场景下维持高效搜索状态的难度很大。两条路线各有盲区：前者受困于离散-连续耦合，后者受困于人工设计的天花板。详细的理论分析与文献综述将在第二章展开。

LLM 在组合优化领域的应用是近两年的研究焦点。已有工作主要面向网约车调度、旅行商问题（TSP）或装箱问题等离散场景，环境状态可抽象为图结构或线性约束。UAV-MEC 系统则包含受空间位置影响的非凸信道模型和非线性推进能耗模型，二者形成连续时间动力学特性与离散调度决策的深度绑定。如何设计适应双向数据流与连续轨迹规划的 LLM 演化机制，是当前尚未充分验证的问题。

## 1.3 本文主要工作与贡献

本文在 NeurIPS 2025 的 LLM 引导优化框架基础上，针对 UAV-MEC 系统的物理特性进行跨领域改造，提出融合大模型演化与经典数学规划的双层分解求解架构。具体工作与贡献包含以下四个方面：

第一，构建了考虑双向数据流的 UAV-MEC 联合优化模型。面向多架固定高度旋翼 UAV 服务地面终端的场景，完整刻画了"上行传输—边缘计算—下行回传"三段串行链路。模型显式处理了上行数据量远大于下行数据量的不对称性，量化了通信链路、计算任务执行与 UAV 空间位移之间的时空耦合关系，以卸载决策 $\mathbf{x}$、CPU 频率分配 $\mathbf{f}$ 与 UAV 轨迹 $\mathbf{q}$ 为优化变量建立了 MINLP 问题（第三章）。

第二，设计了双层分解求解架构。外层（Level-1）使用二进制线性规划（BLP）处理离散的卸载决策；内层（Level-2）在给定卸载方案的前提下，以块坐标下降（BCD）与连续凸近似（SCA）交替求解 CPU 频率分配与轨迹规划。该结构在保证内层收敛性的同时，压缩了外层的组合搜索空间（第四章）。

第三，提出了基于 LLM 引导的目标函数演化机制。以和声搜索（Harmony Search）为外层搜索框架，将 LLM 作为高级变异算子。通过构建包含物理模型特征与当前种群状态的定制化 Prompt，引导 LLM 自动生成代码形式的新一代目标函数，使外层搜索不再受限于固定的人工规则，改善候选解的多样性与质量（第四章）。

第四，开发了完整的仿真平台并完成性能验证。基于 Python 实现了涵盖无线信道模拟、UAV 动力学仿真、双层优化求解器和 LLM 交互接口的测试环境，在不同网络规模与负载条件下评估了所提架构的收敛行为和调度性能（第四章）。

## 1.4 论文组织结构

本文共分五章，各章内容安排如下：

**第一章　绪论。** 阐述 UAV 辅助边缘计算的应用背景与 MINLP 优化问题，概括传统求解路线与 LLM 引导优化的现状及局限，明确本文的工作与章节安排。

**第二章　基本原理。** 介绍移动边缘计算与 UAV 通信的物理基础，梳理非凸优化理论，阐明 LLM 在组合优化问题中作为演化算子的工作机制。

**第三章　系统模型与问题建模。** 构建涵盖空地信道、双向数据流、边缘计算处理和旋翼推进能耗的综合物理模型，形式化定义联合优化的 MINLP 问题 $P_0$。

**第四章　求解算法与实验分析。** 推导 Level-1 离散搜索与 Level-2 BCD/SCA 连续优化的双层解耦过程，阐述基于 LLM 与和声搜索结合的目标函数演化机制，给出仿真实验设置与结果分析。

**第五章　总结与展望。** 归纳全文工作与实验结论，讨论所提架构的适用边界与未来研究方向。
