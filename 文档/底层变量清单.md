# 底层基础变量与配置参数清单

以下整理并总结了所有**不需要由公式推导，且必须由系统配置、物理环境或生成数据集**直接定义的底层基础变量及参数。在代码实现（如 `setting.cfg` 或仿真初始化）中必须优先实例化这些参数。

## 1. 物理环境与网络常量
|               变量名               | 单位 / 描述     | 释义补充                                                                                         |
| :--------------------------------: | :-------------- | :----------------------------------------------------------------------------------------------- |
|           $\mathcal{T}$            | 集合 / 时间范围 | 整个优化的决策时域 $t \in \{1,\dots,T\}$                                                         |
|              $\delta$              | s (秒)          | 单个时隙长度（Time slot duration）                                                               |
|        $x^{\max}, y^{\max}$        | m (米)          | 空域和地面的最大边界范围                                                                         |
|                $H$                 | m (米)          | 无人机固定飞行高度                                                                               |
|               $N_0$                | Watt (W) 或 dBm | 环境噪声功率或功率谱密度                                                                         |
|       $d_U^{\mathrm{safe}}$        | m (米)          | 无人机间避免碰撞的最小安全距离约束                                                               |
| $B_{i,j}, B_{j,i}^{\mathrm{down}}$ | Hz              | 设备间的正交上行与下行频带带宽常数                                                               |
|            $g_{i,j}^t$             | 标量            | 通信信道增益（通常在仿真模型内部由距离等通过自由空间损耗公式求得，对调度模型而言是已知输入状态） |

## 2. 任务生成属性（输入数据集）
对于每个时隙生成的任务 $U_i^t$，包含了以下不可分割的属性约束：
|   变量名    | 单位 / 描述       | 释义补充                                     |
| :---------: | :---------------- | :------------------------------------------- |
|   $D_i^l$   | bits              | TD 生成需要上行传输到边缘节点的原始数据量    |
|   $D_i^r$   | bits              | 边缘节点计算完成后需要回传给 TD 的结果数据量 |
|    $F_i$    | CPU cycles        | 此任务完成所需的所有 CPU 并发计算工作总量    |
|  $\tau_i$   | s (秒)            | 此任务执行的全流程所允许容忍的最大时间限度   |
| $\zeta_i^t$ | 指示变量 (0 或 1) | 表示终端设备 $i$ 在时隙 $t$ 是否有新任务生成 |

## 3. 设备与边缘节点的硬件规格
|              变量名              | 单位 / 描述   | 释义补充                                                           |
| :------------------------------: | :------------ | :----------------------------------------------------------------- |
|       $\gamma_i, \gamma_j$       | 标量参数      | 决定 TD 或边缘节点芯片执行计算时有效电容特征的能耗系数             |
|             $f_i^t$              | Hz (cycles/s) | 移动设备的本地可用最大计算能力频率                                 |
|          $P_i, P_j^t$            | Watt (W)      | TD 上行发射功率（常数）及边缘节点下行的无线电传输发射功率          |
|           $v_U^{\max}$           | m/s           | 无人机飞行硬件允许的最高移动速度上限                               |
|       $v_j^{\mathrm{tip}}$       | m/s           | 旋翼无人机的叶尖旋转速度常量（决定推进功率大小）                   |
| $\eta_1, \eta_2, \eta_3, \eta_4$ | 均值常数      | 四个与无人机结构、电机效率有关的气动参数系数，均是定值             |
|          $E_j^{\max}$            | Joule (J)     | 边缘节点 $j$ 的总能耗归一化基准（能耗约束上限）                    |

## 4. 待优化的核心决策变量（算法目标）
虽然这部分变量也是构成总公式的底层因子，但它们是您的**优化模型/启发式算法**（比如 `harmony search`、`milpModel`）去动态决定（调整）寻找最优解的”自变量”：

### 4.1 独立优化变量

- **离散卸载决策：**
  - $x_{i,i}^t \in \{0,1\}$：本地决策变量，表示任务 $U_i^t$ 由设备 $i$ 在本地自行处理的二元决策。
  - $x_{i,j}^t \in \{0,1\}, j \in \mathcal{J}$：卸载决策变量，表示任务 $U_i^t$ 被卸载到边缘节点 $j$ 执行。
- **连续资源分配：**
  - $f_i^t$：设备 $i$ 在时隙 $t$ 的本地 CPU 频率。
  - $f_{j,i}^t$：被选中的边缘服务器 $j$ 分配给任务 $U_i^t$ 的计算频率资源。
- **UAV 轨迹（位置序列）：**
  - $\mathbf{q}_j^t = [x_j^t, y_j^t]^T$：无人机 $j$ 在时隙 $t$ 的二维水平位置。这是唯一的独立轨迹变量。初始与结束位置约束为 $\mathbf{q}_j^I$ 和 $\mathbf{q}_j^F$。

### 4.2 派生量（由独立变量唯一确定，非额外自由度）

- $\mathbf{v}_j^t = (\mathbf{q}_j^{t+1} - \mathbf{q}_j^t)/\delta$：速度向量，由相邻时隙位置唯一确定（见 `公式.md` 公式 3'）。
- $v_j^t = \|\mathbf{v}_j^t\|$：速度模长。
- $P_j^{\mathrm{prop},t}$：推进功率，由 $v_j^t$ 经公式 (18) 计算。
- $E_j^{\mathrm{fly},t} = P_j^{\mathrm{prop},t}\cdot\delta$：单时隙飞行能耗。
- $r_{i,j}^t$, $r_{j,i}^{\mathrm{down},t}$：上行/下行速率，由 $\mathbf{q}_j^t$ 通过信道增益间接确定。
- $D_{i,i}^t = \mu_i^t / f_i^t$：本地执行时延。
- $D_{i,j}^t$：远程卸载总时延，由 $f_{j,i}^t$ 和 $\mathbf{q}_j^t$ 间接确定。

## 5. 算法优化超参数（系统成本权重）
这部分变量不在物理或任务生成模型中存在，但它是构建加权系统成本函数从而进行策略寻优的关键基础设定：
|  变量名   | 描述     | 释义补充                                                         |
| :-------: | :------- | :--------------------------------------------------------------- |
| $\alpha$  | 权重系数 | 归一化总系统成本中关于 **”时延开销”** 的重要性配比权重           |
| $\gamma$  | 权重系数 | 归一化总系统成本中关于 **”边缘节点计算侧能耗”** 的重要性配比权重 |
| $\lambda$ | 权重系数 | 归一化总系统成本中关于 **”无人机飞行推进能耗”** 的重要性配比权重 |
