## 二、公式符号统一

### 触发原因

用户发现草稿公式与 `文档/10_模型与公式/公式与两层解耦整合版_最新版_2026-03-24.md` 不一致。

### 方法

使用 Explore agent 对比两份文档，提炼 14 处差异清单；另用 Explore agent 确认 L1-C2 的代码/文档冲突细节；用 AskUserQuestion 确认 L1-C2 策略（跟代码，标注差异）。

### 修正的 14 处符号差异

| 类型 | 修正内容 |
|------|---------|
| 决策变量 | $x_{\text{local}}^{i,t}$ → $x_{i,i}^t$；$x_{\text{offload}}^{i,j,t}$ → $x_{i,j}^t$ |
| 时延/能耗符号 | 编程记号 `[i][t]` → 数学上标 $\hat{D}_{i,i}^t$、$\hat{E}_{j,i}^{\mathrm{comp},t}$ |
| 权重符号 | $\gamma_w, \lambda_w$ → $\gamma, \lambda$ |
| C1 右侧 | `= 1`（活跃过滤） → `= ζ_i^t` |
| C3 求和 | 不含 $\zeta_i^t$ → 含 $\zeta_i^t$（与参考文档一致） |
| UAV 轨迹 | $q_j^t$ → $\mathbf{q}_j^t$（粗体向量） |
| 频率下标 | $f_j^{i,t}$ → $f_{j,i}^t$ |
| $N_{\text{fly}}$ 定义 | $\|\mathcal{J}\|\times(\|\mathcal{T}\|-1)$ → $\|U\|(T-1)$ |

### L1-C2 差异说明

参考文档（2026-03-24）未设 L1-C2 硬约束；代码（2026-04-04 后）加入了硬约束（超期卸载置零）。用户确认论文跟代码走，在约束说明后添加了差异备注。
