#### D2：Phase⑥ Step 3 求解器集成诊断 ❌ **发现致命问题**
**问题定位**：
```
DCP check: False
ERROR: DCPError: Problem does not follow DCP rules.
The following constraints are not DCP:
var4 == quad
```

**根本原因**：trajectory_opt.py:527-533 的通信时延约束非 DCP
```python
rate_safe = cp.maximum(rate_lb_expr, 1e-12)
dist_from_height = cp.sqrt(z)
delay_ub = 2.0 * dist_from_height * cp.inv_pos(rate_safe)
constraints.append(delay_ub <= tau_comm_budget + 1e-9)
```

**后果**：所有 CVXPY 求解器（CLARABEL/ECOS/SCS）都拒绝，返回 None，测试 12/12 失败

**关键发现**：这不是求解器配置问题，是**约束表达式本身设计错误**
