## 📝 核心内容概览

### 问题根源（两层）

**Level 1（理论）**：原始联合优化 $\min_{x,f,P,q} C(x,f,P,q)$ 是混合整数非凸的
- 已有文档：`公式20_凸性分析.md`
- 对策：BCD分解，固定 $x,f,P$ 优化 $q$

**Level 2（实施）**：轨迹子问题 $\min_q \phi(q)$ 理论凸但CVXPY代码违反DCP规范
- 违规1：通信延迟约束 `sqrt(dist) × inv_pos(rate)` 是凹×凸乘积 ❌
- 违规2：安全分离约束 SCA线性化多次迭代导致约束集非凸 ❌

### SOCP修复方案（推荐）

**转换思路**：
```
原约束（非凸）：
  2·√(H² + ||q_j - w_i||²) · (1/rate) ≤ τ_comm

改写为SOCP（DCP规范）：
  (SOCP-1)  ||[H, (q_j - w_i)_x, (q_j - w_i)_y]||₂ ≤ s_ji
  (SOCP-2)  2·s_ji·rate ≥ D_i
```

**优势**：
- 完全DCP规范 ✓
- CLARABEL/ECOS/SCS直接求解 ✓
- 无近似误差，数学精确 ✓
- 论文标准形式 ✓
