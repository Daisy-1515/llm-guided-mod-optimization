## 目录约定

- 每个 `uX/` 子目录对应固定 `numUAVs = X`
- 子目录下保留原始时间戳目录，不改动内部结果文件
- 有重复时间戳时，说明该点位历史上跑过多次，后续汇总时需要明确选用哪一版
